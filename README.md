# Varga Photography Website

A modern, mobile-friendly photography portfolio website showcasing travel, nature, and portrait photography. Built with HTML, CSS, and JavaScript.

## 🌍 Live Site

https://andris811.github.io/vargaPhoto/

## ✨ Features

### 🖼️ Gallery
- Category filtering: Nature, Portrait, Landscape, Street, Film, B&W
- Full-screen lightbox viewing with keyboard navigation
- Right-click protection on all images

### 🌐 Multilingual
- English, Hungarian, Chinese
- Auto-detects browser language

### 🌗 Light/Dark Mode
- System preference detection
- Manual toggle with persistence
